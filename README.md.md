# Retro Breakout Arena

O implementare modernă a clasicului joc Breakout, dezvoltată în C++ folosind un framework grafic bazat pe OpenGL.

Caracteristici:
- Sistem de coliziuni sferă-dreptunghi pentru bilă și paletă.
- Generare procedurală a cărămizilor cu diferite rezistențe.
- Power-up-uri și efecte vizuale pentru distrugerea obiectelor.
- Control fluid al paletei folosind tastatura.
